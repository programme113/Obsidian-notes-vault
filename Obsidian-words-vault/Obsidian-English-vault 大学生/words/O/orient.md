# 词义
- 英：/ˈɔːrient/； 美：/ˈɔːrient/
- #vt 朝向；适应；使适应；面对；熟悉；确定方向；确定方位；认识方向
- #n 东方；亚洲；东洋；远东
- #v 使朝向；适应；使面对；确定方向
- #adj 东方的；新生的；(太阳等)上升的；东洋的
# 例句
- Sets a plane using a point that lies within it plus a normal to orient it ( note that the normal must be a normalised vector ) .
	- 设置平面上的点和法线的朝向（注意法线必须是规范化的向量）。
- A target - oriented line search method for automatic circuit layout on minicomputers
	- 朝向目标的线探索法&用小型计算机实现自动布线
- It took him some time to orient himself in his new school .
	- 他经过了一段时间才熟悉新学校的环境。
