# 词义
- 英：/kəʊd/； 美：/koʊd/
- #n 代码；编码；密码；法典；行为规范；电码；道德准则；暗码
- #vt 编码；为…编码；编程序；把…译成密码
# 例句
- A special code is needed to override the time lock .
	- 这定时锁要用特定密码才能打开。
- Tap your code number into the machine .
	- 把你的密码数字输入机器。
- The bar codes on the products are read by lasers .
	- 产品上的条形码是用激光读取的。
